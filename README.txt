How to open it locally (choose ONE):

A) VS CODE — Live Server (easiest)
1) Open VS Code.
2) File → Open Folder… → select the 'my-local-site' folder.
3) Install the extension "Live Server" by Ritwick Dey (one-time).
4) In VS Code, open index.html and click the "Go Live" button (bottom right), 
   or right–click index.html → "Open with Live Server".
5) Your browser will open to something like: http://localhost:5500
   Every time you save, the page auto‑reloads.

B) Python (no extensions)
1) Open a terminal in this folder.
2) Run ONE of these (depends on your system):
   - python3 -m http.server 5500
   - py -m http.server 5500   (Windows)
3) Visit http://localhost:5500 in your browser.
